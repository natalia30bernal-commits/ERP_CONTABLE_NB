# ERP CONTABLE NB - Netlify + Supabase

Proyecto React/Vite listo para subir a Netlify.

## Publicar en Netlify

1. Descomprime este ZIP.
2. Sube la carpeta completa a GitHub o arrástrala a Netlify.
3. En Netlify usa:
   - Build command: `npm run build`
   - Publish directory: `dist`
4. Agrega variables de entorno:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`

## Supabase

Ejecuta el archivo `supabase_schema.sql` en SQL Editor de Supabase.
